 TASK 1: Automate Code Deployment Using CI/CD Pipeline (GitHub Actions)
 Objective: Set up a CI/CD pipeline to build and deploy a web app.
 Tools: GitHub, GitHub Actions, Node.js, Docker
 Deliverables: GitHub repo with .yml CI/CD workflow
 
 Hints / Mini Guide:
 1. Use .github/workflows/main.yml to define the pipeline.
 2. Configure jobs for build and deployment.
 3. Use DockerHub for image deployment.
 4. Automate steps like test → build → push.
 5. Test using a sample Node.js app.
 6. Trigger on push to main.
